# Sample Inquiry

## Example Customer Submission

```text
Name: Sofia Wagner
Email: sofia.wagner@example.com
Cruise Type Preference: Ocean
Destination / Route: Mediterranean
Travel Period: July 2026
Number of Travelers: 2
Message:
We are looking for a Mediterranean cruise in July for 2 adults with a balcony cabin.
Flights included would be ideal. Please send suitable options.
```

## Why This Example Matters

This inquiry is considered **high quality** because it includes several useful travel details:

- Destination
- Travel period
- Number of travelers
- Cruise type preference
- Cabin preference
- Explicit request for offers

These attributes allow the AI classifier to make a strong decision about:

- `inquiry_quality`
- `travel_intent`
- `next_action`
